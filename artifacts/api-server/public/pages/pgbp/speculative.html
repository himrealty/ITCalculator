📋 Important Rules for Speculative Business:
  • Speculative losses can ONLY be set off against speculative profits
  • Non-speculative losses (F&O) can be set off against any business income
  • Both types can be carried forward for 4 assessment years
  • Speculative: Intra-day equity, commodity futures, etc.
  • Non-Speculative: F&O, currency derivatives, etc. (treated as business income)
